# SOC3070 Análisis de Datos Categóricos 

Éste repositorio contiene el material del curso SOC3070 Análisis de Datos Categóricos, dictado el segundo semestre 2020 a estudiantes de postgrado del Departamento de Sociología de la Universidad Católica de Chile.

## Material del curso

### Nivelación

Una plantilla de código que cubre las funciones típicamente usadas en un proyecto real de intestigación. El código tiene la estructura de lo que considero un workflow efectivo y ordenado. Consiste en un [`[master file]`](pre_calculo.pdf) desde el que llamamos una serie de `Rscript` que implementan diferentes partes del análisis: [`[exploración]`](pre_calculo.pdf) y [`[análisis]`](pre_calculo.pdf) de datos, incluyendo modelos de regresión, producción de tablas y gráficos publicables.
